# CommonMark Timeline

[[TOC]]

## Introduction

A lightweight and flexible **League CommonMark extension** that adds support for the `::: timeline` fenced block.

## Instalation

You can install the package via composer:

```bash
composer require zaimea/commonmark-timeline
```

or via composer.json

```json
"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/zaimealabs/commonmark-timeline"
        }
    ]
```
